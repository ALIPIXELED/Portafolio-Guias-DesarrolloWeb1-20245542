Guías de Ejercicios

Guía 1: Introducción y conceptos básicos de HTML y CSS

En esta guía, el estudiante aprenderá a identificar la estructura básica de un documento web, aplicar estilos CSS mediante estilos embebidos y hojas externas, y utilizar selectores y propiedades CSS para dar formato a fuentes y textos.

Ver Ejercicios

Guía 2: Selectores CSS avanzados y modelo de caja

En esta guía, el estudiante aprenderá a utilizar y combinar selectores CSS avanzados, comprenderá la función de las pseudo-clases y pseudo-elementos, y aplicará el Modelo de Cajas para diseñar páginas web, identificando y manejando sus componentes: margen, borde, relleno y contenido.

Ver Ejercicios

Guía 3: Posicionamiento, visualización y maquetación de sitios web

En esta guía, el estudiante aprenderá a utilizar los distintos tipos de posicionamiento en CSS y a diferenciarlos según su uso, aplicará propiedades relacionadas con la visualización de elementos y diseñará maquetas web con enfoques tanto fijos como fluidos.

Ver Ejercicios

Guía 4: Introduccion al Responsive Design

En esta práctica, el estudiante aprenderá a crear y estilizar formularios con los controles de HTML5, utilizará adecuadamente la metaetiqueta `viewport`, definirá media queries según el ancho del navegador y aplicará todos los conocimientos adquiridos para desarrollar páginas web responsivas.

Ver Ejercicios
